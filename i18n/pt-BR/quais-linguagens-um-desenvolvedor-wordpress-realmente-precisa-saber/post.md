Trabalhar com WordPress vai muito além de instalar tema e plugin. Se a ideia é atuar como desenvolvedor (e não apenas usuário avançado), existem requisitos técnicos mínimos que você precisa dominar — e isso não significa saber tudo, mas saber o suficiente para entender, modificar e criar com segurança e autonomia.

🔹 HTML + CSS
Base de toda a estrutura e estilo. Saber estruturar conteúdo, aplicar estilos e entender boas práticas é o primeiro passo.

🔹 PHP
WordPress é construído em PHP. Mesmo que você não crie plugins do zero, vai precisar entender como os temas e funções funcionam por trás dos bastidores.

🔹 JavaScript (básico ao intermediário)
Muitos temas modernos usam JS para interatividade. E com o editor de blocos (Gutenberg), o conhecimento em JS se tornou ainda mais relevante.

🔹 MySQL (básico)
Saber como o banco de dados do WordPress é estruturado ajuda na criação de queries personalizadas, filtros e otimizações.

🔹 WordPress Core, Hooks e Template Hierarchy
Saber o que é um filter, action, como funciona o functions.php, a hierarquia de templates… é o que separa um customizador de um desenvolvedor.

🔹 Git + boas práticas de versionamento
Se vai colaborar em projetos ou manter um portfólio profissional, saber Git é essencial.

🔹 Conhecimento mínimo de segurança e performance
Não precisa ser DevOps, mas precisa evitar vacilos: SQL Injection, sanitização, validação, uso correto de escopos e recursos nativos.
